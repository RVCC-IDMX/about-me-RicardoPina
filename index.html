Animated Scalable Vector Graphics

Animated Scalable Vector Graphics (SVG) is a type of vector graphic that can be used to create interactive, animated graphics for websites and applications. SVG is an XML-based vector image format that can be used to create high-quality, resolution-independent graphics. Creating an animated SVG is relatively simple. First, you need to create the vector graphic in a vector graphics editor such as Adobe Illustrator or by coding it yourself –if you are crazy, that is. Once the graphic is created, you can then export it as an SVG file. The next step is to add and create the animation using SVG elements. SVG Paths and CSS The path element is a basic shape that may be used to create almost any advanced 2D shape you can imagine. To animate, we can take advantage of two SVG attributes: stroke-dasharray and stroke-dashoffset. The stroke-dasharray attribute is used to define the pattern of dashes and gaps that will be used to draw the path. The value of this attribute is a list of comma-separated numbers that represent the lengths of the dashes and gaps. For example, if you wanted to create a dashed line, you could use the following value: stroke-dasharray: 5,5. This would create a pattern of five-pixel dashes and five-pixel gaps.

Animation or Optical illusion?

Suppose we wanted to animate a simple bezier curve as if it was being drawn gradually and smoothly on the screen. we will have to set the dash (and gap) lengths, using the stroke-dasharray attribute, equal to the path length. This is so that the length of each dash and gap in the dashed curve is equal to the length of the entire path.

Next, we will set the dash offset, using the stroke-dashoffset attribute, to 0. This will make the path appear on the screen as a solid curve (we are essentially looking at the first dash, and we already made each dash span the entire length of the curve). By setting the dash offset equal to the length of the curve, we will end up with an invisible curve (we are now looking at the curve being rendered as an entire gap—the part that immediately follows a dash).

Now by animating the stroke-dashoffset property, you can make the curve appear on the screen gradually.
